# Mini Editor

## Featureless editor...

Create config file in this directory:
`$HOME/.config/mini/config.json`
with next content 
```
{
	"config": {
		"font_name": "font_name",
		"font_size": 10,
		"color_scheme": "dark"
	}
}
```

How to compile?

```
$ git clone  github.com/iwannajump/mini`
$ cd mini
$ cmake .
$ make
```
